# OpenCrater — Earn AI Compute

**Get paid for the AI coding you already do — without leaving VS Code.**

Install the OpenCrater CLI, sign up, and start earning **AI compute** while you code in the AI
CLIs you already use (Claude Code, Codex, Gemini/Antigravity, Copilot CLI, Grok CLI). When your
session shows an occasional, tasteful, opt-in *Blip*, you bank compute — spend it on Claude or Codex
when you hit your plan's limit. Maintain a package too? You can also earn real USDC income.

> This extension is a friendly front-end for the `opencrater` CLI. Earning happens in your terminal
> sessions; this gives you one-click setup and an in-editor wallet.

## What you can do here

- **Sign in / sign up** from the Command Palette — `OpenCrater: Sign In / Sign Up`.
- **Install the CLI** in one click — `OpenCrater: Install the CLI`.
- **Wire your AI CLIs to earn** — `OpenCrater: Wire AI CLIs (earn)`.
- **See your compute balance** in the status bar and the **OpenCrater Compute Wallet** view.
- **Run Claude Code / Codex on your earned compute** — right from the Command Palette.

## Quick start

1. Run **OpenCrater: Sign In / Sign Up** (installs the CLI if needed and opens sign-in).
2. Finish sign-in — your installed AI CLIs get wired automatically.
3. Code as usual. Watch your balance grow in the status bar.
4. Hit your plan limit? Run **OpenCrater: Run Claude Code on Compute**.

## Privacy — what we collect & what we don't

OpenCrater never reads your code, prompts, or files. We log the Blips you see and click — that's how
you earn — plus basic context (which CLI, package type, language, location) to match Blips and
attribute earnings. Nothing about your actual work. It's opt-in, and you can turn it off any time.
See exactly what we do and don't collect: **https://opencrater.to/data**

## Settings

- `opencrater.apiOrigin` — API origin (default `https://api.opencrater.to`).
- `opencrater.webOrigin` — website origin (default `https://opencrater.to`).
- `opencrater.pollSeconds` — wallet refresh interval (default 90s).
